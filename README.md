# KidsPOS Server

キッズビジネスタウンで使用するPOSシステムのサーバーサイドシステムです。

## Raspberry Pi向け最適化

このアプリケーションはRaspberry Pi上で動作することを想定しています。メモリ使用量を最適化するための各種設定が実装されています。

### このプロジェクトの最適化内容

1. JVMメモリ設定
   - 起動スクリプトでヒープサイズを最適化
   - G1GCガベージコレクタの使用

2. Spring Boot設定の最適化
   - Tomcatスレッド数の削減
   - 不要なAutoConfigurationの無効化

3. キャッシュ機構の導入
   - 頂幻にアクセスされる商品やスタッフデータのキャッシュ

4. 非同期処理
   - レシート印刷のバックグラウンド実行

5. メモリ使用状況のモニタリング

### 起動方法

1. このリポジトリをクローン
2. gradleでビルド: `./gradlew bootJar`
3. `start.sh`を実行可能にする: `chmod +x start.sh`
4. 起動スクリプトを実行: `./start.sh`

Raspberry Piのモデルによっては、`start.sh`ファイルのJVMメモリ設定を調整することをお勧めします。
